CSS

Why CSS was created

_ Design & Control: It gives you total power over how your site looks—colors, fonts, spacing, and layouts. Without it, every website would look like a boring 1990s research paper.

_ Efficiency: You can style 1,000 pages at once by linking them to a single CSS file.

_ Responsiveness: It allows your website to adapt for different screen sizes like mobile and desktop.

_ User Experience (UX): CSS improves interaction like hover effects, animations, and layout clarity.

Road map for learning

_ Foundations: Selectors, Specificity, Box Model (Margin, Border, Padding).

_ Typography & Colors: Fonts, alignment, Hex & RGB colors.

_ Layouts: Flexbox and CSS Grid.

_ Positioning: relative, absolute, fixed.

_ Responsiveness: Media Queries.

_ Animations
